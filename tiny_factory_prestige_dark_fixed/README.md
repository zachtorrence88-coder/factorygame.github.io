# Tiny Factory Prestige Dark

Updated GitHub + Cloudflare Pages ready static site.

## Fixes
- Conveyor belts now move into:
  - other belts
  - fast belts
  - splitters
  - machines that can accept items
- Unlocks moved **below** the main play area
- Build buttons are now color-coded
- Dark mode is default
- Light mode toggle added

## Deploy
1. Put these files in the root of your GitHub repo
2. Connect the repo to Cloudflare Pages
3. Use **no framework**
4. Leave the build command blank

## Notes
- No `_redirects` file is included
- `_headers` is included for iframe embedding
